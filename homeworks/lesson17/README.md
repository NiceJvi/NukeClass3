# 第17课作业
---
请制作一个 Gizmo，其中包含两个 Text 节点。
在 Gizmo的 参数设置面板上，支持为其中一个 Text 节点设置文本，
以及同步调节两个 Text 节点的字体字号。Gizmo 上另有三个按钮，
分别支持将文本拷贝到另一 Text 节点，
以及从外部的某个 Text 节点拷贝文本到内部的某个 Text 节点（用下拉菜单选择拷贝到哪个 Text），
还有将 Gizmo 转换成 Group。
